## ğŸ® One-Minute Summary

**Connect 5** is a turn-based 3D strategy game where two players stack 1Ã—2 blocks to build 5 in a row and win.

### Quick Facts
- **Backend**: Python Flask REST API + WebSocket
- **Frontend**: Angular with Three.js 3D rendering
- **Network**: LAN-capable multiplayer
- **Setup Time**: ~5 minutes
- **Code**: ~1000 lines, well-documented

### To Start
```powershell
.\start.ps1 setup          # Once
.\start.ps1 run-all        # Then
# Visit http://localhost:4200
```

### To Play
1. Player 1: Create game
2. Player 2: Join game
3. Take turns clicking positions
4. First to 5 in a row wins
